## 두 리스트 합치기

#### 배열 합치기 

sort 이용

````python
m = [1,3,5,7,9]
n = [2,4,6,8]

num = m + n #[1,3,5,7,9,2,4,6,8]
num.sort() = [1,2,3,4,5,6,7,8,9]
````
